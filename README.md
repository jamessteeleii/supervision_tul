# A comparison of time under load during resistance training under supervised and unsupervised conditions in a sample of strength training clinic members
This repository includes the materials, data, and analysis code for a trial examining the effects of resistance training with and without supervision in Kieser members upon time under load when using their existing training loads and instructed to train to momentary failure.
